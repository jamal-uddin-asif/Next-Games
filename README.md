# Next Games

## Description

This application is for browsing games, you can explore games, and see details, if your logged in you can add your game in our application.

## Route Summary
- Home: User can see latest games in home page
- Games: User can all games in Games route
- Add Game: User can add there game
- Manage Games: User can manage games
